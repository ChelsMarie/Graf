At the top of the page you'll see the toolbar. There are multiple tools to select

The default tool is the Select tool, which can be used to select the nodes and edges on the graph.

Use the Node tool to add nodes to the graph.

The edge tool can be used to add both directed and undirected edges.

The label tool can be used to label nodes and edges.

The erase tool is used to erase nodes and edges.

The algorithm tool is used to apply algorithms to the graph, such as Dijkstra's algorithm. Simply select two nodes with the select tool, then select which algorithm to apply.

Happy Grafing!
